# tumamoca_genome_assembly
Tumamoca (GlobeBerry) Genome Assembly reproducible HiFi + Hi-C genome assembly workflow, including QC, organelle filtering, contig curation, Hi-C scaffolding, polishing, and assembly evaluation (BUSCO &amp; Merqury). This repository tracks scripts, configs, and summary outputs used to build and validate the first reference genome for Tumamoca.
